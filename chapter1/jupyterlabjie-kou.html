Jupyterlab接口文档

一、接口简介

将数据增加到Mysql数据库里面的models表和user_model_type表以及模型详情信息 --- （Add_model.py）

返回前端json数据---模型分类和基础信息 ---- （Type_handler.py）

二、接口详情

  • 请求地址

'/model/add' ----（Add_model.py）

'/model/type' ---(Type_handler.py)

  • 请求类型

get、post

  • 请求参数

/model/type请求参数：无

/model/add 请求参数：

参考名 类型 必填 描述 默认值 参考值
Model_Name 整形 是 模型名字 - model测试1
Code 字符串 是 模型代码 - print("520520")
Permission 整形 是 权限 - 1
Desc 字符串 是 对该模型的描述 - 输出520520
ctype_id 整形 是 子类模型ID - 2
user_id 整形 是 用户ID - 1

（Add_model.py）

  • 返回数据

1：'/model/add'

get方法：return model（对象）

返回前端json数据

post方法：return 状态和状态吗

添加成功返回： Model added successfully

添加失败返回：HTTP状态码 400

2：'/model/type'

get方法：返回前端json数据。
